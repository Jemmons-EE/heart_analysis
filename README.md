# heart_analysis
Heart Data Analysis;

Simple Example of Machine Learning analysis through Python and SQL (sqlite3); 
Linear-Imperative Script;

Pulls RAW data file from Github and reads into CSV / SQL; 
Machine Learning analysis and Prediction through 6 algorithms - Perceptron, Logistic Regression, SVM, Decision Tree, Random Forest, KNN -;
Outputs results to SQL table and converts to CSV for transfer out;
